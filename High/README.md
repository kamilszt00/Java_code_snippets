Program is a simple too high to low game.
Player will be first asked to provide the range of the number.
Range needs to be a positive numbers, and length of it needs to be bigger than 3.

Then machine will pick one number from the provided range, and the guessing game will start.
Player will be asked for a guess, each guess must be checked if:
- it is a number that machine picked
- it is bigger than number picked by machine
- it is smaller than number picked by machine
- IF it is even a number
- IF the number is in the range picked by player
Game will be on as long as the number isn't guessed or if player type "Exit".

If the guess passed all conditions, increase the number of guesses made by player and proceed to the next step of the game


After player guess of exit the game or win, machine will provide player with score.
Score consists of number of guesses and grade based on number of guesses.

Grades:
- 1 - cheater
- 2-10 - probably a cheater
- 10-20 - acceptable
- 20-30 - noob
- 30-40 - nub
- 40+ - a troll
